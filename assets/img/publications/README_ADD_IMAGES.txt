Publication figure slots
========================

How the page looks for images:
- The site first tries: assets/img/publications/<paper-id>.png
- If that file is missing, it tries: assets/img/publications/<paper-id>.jpg

Examples:
- assets/img/publications/percolation-polydisperse-nanorods-2020.png
- assets/img/publications/vitrimer-topological-transition-2024.jpg
- assets/img/publications/ml-tg-sbr-2026.png

Recommended image choice:
- Graphical abstract
- TOC / cover-style figure
- The most recognizable summary figure from the paper
- Crop it cleanly to a wide rectangle when possible

Recommended size:
- 1400 px wide or larger
- PNG preferred; JPG also works

No JavaScript edits are needed.
Once the file is placed in this folder with the correct paper ID, the publication card will load it automatically.
