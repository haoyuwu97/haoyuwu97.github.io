Haoyu site v7 — minimal patch notes

This patch is built on the latest multi-page version and focuses only on the requested areas.

Included fixes and upgrades
1. Publication figures
- Bundles the entire local publication image folder into the patch.
- Replaces site-data.js with a valid, fully formatted version so publication data loads correctly.
- Hardens figure rendering in renderers.js with robust absolute asset resolution and fallback logic.

2. Background rendering
- Replaces the background engine with a more polymer-like bead-spring field.
- Adds wormlike, semiflexible, folded, ring, and brush-like architectures.
- Adds pendant side groups for brush-like chains.
- Uses lower-frequency contour modes and gentle anchor drift instead of rigid translation.
- Keeps faux-3D depth and 3D bead rendering.

3. Research Wiki page
- Changes package pagination to 6 per page.
- Adjusts the repo grid so all 6 packages fit in a cleaner desktop layout.

4. Rest Mode
- Rebuilds the modes toward slower, clearer, and more educational mechanics.
- Living Chain Growth: active-end polymerization with monomer capture, quenching, and catalyst restart.
- Brush Channel Transport: tracer transport through breathing grafted brushes.
- Vitrimer Network Triage: connectivity-preserving bond-repair puzzle.
- Reaction Match Lab: polymer-chemistry matching board.
- Percolation Circuit: rotate-to-connect transport puzzle.
- Keeps the UI small and floating above the full-screen canvas.
