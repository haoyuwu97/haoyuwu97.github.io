# haoyuwu97.github.io

A lightweight personal academic homepage for GitHub Pages.

## Files

- `index.html` — the site structure and content
- `styles.css` — layout, typography, responsive design, and theme styling
- `script.js` — dark/light mode toggle and footer year
- `favicon.svg` — simple vector favicon
- `.nojekyll` — disables Jekyll processing for a plain static site

## Recommended deployment

### Option A: user site

Create a repository named `haoyuwu97.github.io`, place these files at the repository root, and publish with GitHub Pages.

### Option B: project site

Create a normal repository such as `haoyu-homepage`, place these files at the repository root, and publish the repository root with GitHub Pages.

## Editing guide

- Update the hero biography in `index.html`
- Add or remove repository cards in the `Selected repositories` section
- Modify colors or spacing in `styles.css`
- Replace external links in the `Profiles and research pages` section as needed

## Notes

This version is intentionally dependency-free: no build step, no framework, and no external asset pipeline. It is meant to be easy to read, edit, and deploy.
