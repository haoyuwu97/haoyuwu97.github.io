    Design notes
    • No login, no player ID, and no leaderboard: the page is intentionally low-friction and self-contained.
    • Each mode is paced slower than a conventional arcade game so the polymer / chemistry analogy stays visually readable.
    • The aim is not realism but a clean conceptual bridge: chain growth, steric windows, defect fields, crowding, and reactive pathways.
